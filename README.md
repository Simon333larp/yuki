# 🔢 Guess the Number Game

Классическая консольная игра «Угадай число» с подсчётом попыток.

## 🧩 Правила
Компьютер загадывает число от 1 до 100. Ваша задача — угадать его, получая подсказки **«больше»** или **«меньше»**.
